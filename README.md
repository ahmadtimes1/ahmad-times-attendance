# Ahmad Times Attendance & Wages

Online attendance and daily wage system for Ahmad Times For Building Maintenance Company.

## Files

- `index.html` - app screen
- `styles.css` - design
- `app.js` - app logic
- `supabase-config.js` - Supabase project connection
- `supabase-schema.sql` - database setup
- `ONLINE-SETUP.md` - setup guide
- `vercel.json` - Vercel static hosting config

## Deploy

Use Vercel with this folder as a static site.
